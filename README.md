# 🚀 Udemy Clone Website

A clean and modern **Udemy Homepage Clone** developed using **HTML5** and **CSS3**. This project focuses on building a responsive and visually appealing learning platform interface by recreating the design of the Udemy homepage.

## 🌟 Overview

This project demonstrates my understanding of front-end web development concepts including page layout, Flexbox, reusable components, and responsive design principles.

## 🔥 Highlights

- 🎨 Modern UI inspired by Udemy
- 📚 Recommended Courses Section
- ⭐ Popular Courses Section
- 🔍 Search Bar
- 🧭 Navigation Menu
- 🏷️ Topics Section
- 📱 Responsive Layout using Flexbox
- 🦶 Professional Footer
